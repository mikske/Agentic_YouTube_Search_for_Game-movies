# 🎮 YouTube Game Movie Agent

Агент для поиска игрофильмов без комментариев на YouTube.

## Что умеет:

- Поиск видео через YouTube Data API
- Интерпретация запроса (LLM, meta-llama/Llama-3.1-8B-Instruct)
- Фильтрация "без комментариев"
- Ранжирование по релевантности
- Диалоговый интерфейс через Gradio
- Кликабельные ссылки на видео

---

## Архитектура

Pipeline агента:
- User Query

- LLM (или парсер)

- Генерация поисковых запросов

- YouTube API

- Сбор кандидатов

- Скоринг (ключевые слова + длина + просмотры)

- Ранжирование

- Ответ + ссылки

## Пример запроса
Найди игрофильм по PRAGMATA без комментариев

## Почему это агент

Система соответствует паттерну агента:

- понимает цель пользователя
- строит план (несколько поисковых запросов)
- вызывает внешний инструмент (YouTube API)
- анализирует результаты
- возвращает осмысленный ответ

## Примечания
YouTube API имеет ограничения по квоте
